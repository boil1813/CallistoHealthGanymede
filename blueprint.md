# Callisto Health - 개발 청사진 (Blueprint)

## 1. 프로젝트 개요 (Project Overview)
**이름:** Callisto Health (가칭: FitMate - 핏메이트)
**목적:** 친구가 사용자의 체중 감량 목표 달성을 돕기 위해 식단과 운동 루틴을 계획하고 관리해주는 협력형 웹 애플리케이션. 친구는 "매니저" 또는 "코치" 역할을 수행합니다.
**타겟 사용자:** 다이어트를 위해 다른 사람(친구, 연인 등)과 책임감을 공유하고 싶은 사용자와 그들의 페이스메이커.

## 2. 디자인 및 아키텍처 개요 (Design & Architecture Outline)
*   **아키텍처:** 프레임워크 없는 단일 페이지 애플리케이션 (SPA). HTML5, 모던 CSS (Baseline 기능), 바닐라 자바스크립트 (ES Modules, Web Components) 사용.
*   **UI/UX 디자인 가이드라인:**
    *   **테마:** 깔끔하고 모던하며 에너지 넘치는 느낌. 부드러운 노이즈 텍스처가 있는 밝은 배경과 활기를 주는 강조 색상(예: 활기찬 틸(청록색) 또는 생기 있는 코랄색)을 사용하여 사용자의 동기를 부여합니다.
    *   **레이아웃:** 카드 기반의 대시보드 레이아웃. CSS Grid와 Flexbox를 사용하여 모바일에 완벽하게 반응하도록 설계.
    *   **타이포그래피:** 가독성이 뛰어난 산세리프 폰트 (예: 'Inter' 또는 시스템 기본 폰트) 적용. 칼로리와 체중 등 중요한 숫자는 크고 굵은 히어로 텍스트로 강조하여 정보의 위계를 명확히 합니다.
    *   **시각 효과:** 카드 UI에 부드럽고 깊은 다중 레이어 그림자(Drop shadow)를 적용하여 화면에서 "떠 있는(lifted)" 듯한 입체감을 줍니다. 버튼 등의 상호작용 요소에는 호버 시 은은한 "글로우(glow)" 효과를 적용합니다.
*   **핵심 기능 (프로토타입):**
    *   **메인 대시보드 (요약):** 일일 섭취 칼로리(식단) 대비 소모 칼로리(운동) 및 주간 체중 감량 목표 진행 상황을 한눈에 보여주는 위젯.
    *   **식단 관리자 (Diet Manager):** 친구가 계획한 식사 메뉴를 추가하고, 사용자가 식사 후 체크하여 완료 처리할 수 있는 섹션.
    *   **운동 루틴 (Exercise Routine):** 그날의 운동 계획 (예: "30분 조깅", "근력 운동 4세트")을 상세히 보여주고 체크할 수 있는 섹션.
    *   **진행률 추적 (Progress Tracker):** 시간에 따른 체중 변화를 시각적으로 보여주는 그래프 공간.

## 3. 현재 구현 계획 (Phase 1: UI 프로토타입)
*   **1단계: 프로젝트 기본 설정 및 청사진 작성:** 개발의 단일 진실 공급원(Single Source of Truth)이 될 `blueprint.md` 생성. (완료)
*   **2단계: 핵심 HTML 구조 작성:** `index.html`을 수정하여 시맨틱 마크업을 적용하고, 폰트 임포트 및 기본 레이아웃 셸(헤더, 메인 콘텐츠 영역, 모바일용 하단 내비게이션 등) 구성.
*   **3단계: 모던 CSS 기반 작업:** `style.css`에 CSS 변수(색상, 타이포그래피, 간격 등)를 정의하고, 스타일 초기화, 반응형 그리드 시스템 및 카드 유틸리티 클래스 작성.
*   **4단계: 웹 컴포넌트(Web Components) 개발:** `main.js`에 모듈형 컴포넌트 구현:
    *   `<summary-widget>`: 일일 칼로리/체중 요약 위젯.
    *   `<task-list>` / `<task-item>`: 식단과 운동 목록에 재사용 가능한 리스트 컴포넌트.
*   **5단계: 통합 및 목업 데이터 연동:** UI 요소들을 조립하고 현실적인 목업(Mock) 데이터를 입력하여, 친구가 식단과 운동 계획을 입력했을 때 앱이 실제로 어떻게 보이고 작동하는지 시연 가능한 프로토타입 완성.
